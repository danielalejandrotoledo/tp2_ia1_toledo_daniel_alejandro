# Aventura Gráfica Interactiva

Sketch web desarrollado como el segundo trabajo práctico para la materia Informática Aplicada 1, en el marco de la Tecnicatura en Artes Multimediales (UNA).

## Descripción
Proyecto orientado al diseño y desarrollo de una aventura gráfica interactiva que busca reflexionar acerca de la realidad, la virtualidad y la liminalidad, donde mis 3 mascotas —2 fallecidas y una hurtada— se convirtieron en personajes dentro del patio de la casa familiar que actúa como hábitat y a la vez como sepulcro, en el marco del arte póstumo y la tecnología del duelo. Cada botón forma parte de una lógica de estados entre pasado, presente y futuro, y cada cuadro de texto contiene una frase que describe y evoca el efecto o filtro aplicado sobre la imagen.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- p5.js

## Características
- Filtros y efectos visuales e interactivos
- Integración de contenido multimedia generativo
